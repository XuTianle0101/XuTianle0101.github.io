# Academic Personal Homepage

A bilingual academic and career personal site built with Next.js, TypeScript, Tailwind CSS, and MDX.

## Development

```bash
npm install
npm run dev
```

## Content

- Profile and contact data: `src/data/profile.ts`
- Experiences and projects: `src/data/experience.ts`
- Publications: `src/data/publications.ts`
- Interests: `src/data/interests.ts`
- Blog posts: `content/blog/*.mdx`

## Build

```bash
npm run type-check
npm run lint
npm run build
```

The static site is emitted to `out/`.
